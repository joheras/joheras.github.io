Software

  • AntibiogramJ: a Java program to semi-automatically determine, measure and categorise inhibition zones of images from disk diffusion tests.
  • CLoDSA: an open-source image augmentation library for object classification, localization, detection and semantic segmentation.
  • DecoFungi: a web application for automatically predicting the dye decolorisation of fungal strains.
  • DeepClas4Bio: a project that aims to facilitate the interoperability of bioimaging tools with deep learning frameworks.
  • DetectionEvaluationJ: an ImageJ plugin that has been designed to evaluate the performance of object detection algorithms using several metrics.
  • FrImCLa: an open-source framework for Image Classification using traditional and deep learning techniques. It supports a wide variety of deep learning and classical computer vision models.
  • GelJ: a Java program for the analysis of DNA gel fingerprints images.
  • IJ-OpenCV: a Java library that allows the communication of ImageJ/Fiji - a software for image analysis that is widely employed in life sciences - and OpenCV - a well-known computer vision library.
  • YOLO Notebooks: a suite of Jupyter Notebooks to create object detection models using YOLO.
  • Wekabiosimilarity: binary measures implemented in Weka
